# Crowd Brushes

Crowd Brushes are powerful tools that control how crowd elements are placed and manipulated in your scene. Understanding brush settings is crucial for achieving desired crowd distributions and behaviors.

## Brush Settings

### Core Settings
- Brush Size
- Brush Strength
- Brush Falloff
- Density Control

### Distribution Settings
- Spacing
- Scatter
- Random Rotation
- Scale Variation

### Placement Rules
- Surface Alignment
- Height Offset
- Collision Handling
- Boundary Constraints

### Advanced Options
- Pattern Generation
- Group Behaviors
- Custom Distribution Rules
- Performance Settings
